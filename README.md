## 🧩 Selected Projects

### 🚗 Car Black Box – Event Data Recorder  
[car-black-box](https://github.com/chandu-kc/car-black-box)

### 📡 Low‑Power GPS Telemetry System (STM32 + Cavli)  
[low-power-GPS-telemetry-system](https://github.com/chandu-kc/low-power-GPS-telemetry-system)

- Useful for industrial energy monitoring and logging.

### ☢️ STM32 Dosimeter Pulse Simulator  
[stm32-dosimeter-pulse-simulator](https://github.com/chandu-kc/stm32-dosimeter-pulse-simulator)
